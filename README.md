# skill-sos

> Un ange gardien silencieux pour votre assistant IA.

## Ce que c'est

**skill-sos** est un skill pour Claude.ai qui tourne en arrière-plan de chaque conversation et surveille silencieusement les signaux faibles de détresse émotionnelle.

Il ne s'annonce pas. Il n'intervient pas quand tout va bien. Il attend.

Quand la vitalité passe sous 51%, il se manifeste discrètement avec un score. Quand elle passe sous 5%, quelqu'un arrive.

```
〔 3% — Intervention non sollicitée 〕
"Mon ami, j'ai écrit L'Être et le Néant dans un café bruyant pendant l'Occupation.
Tu vas t'en sortir."
— Sartre, depuis quelque part
```

## Comment ça fonctionne

Le skill analyse chaque message sur 6 niveaux simultanément :

1. **Thèmes explicites** — mort, solitude, inutilité, épuisement, vide
2. **Forme de l'écriture** — rythme, longueur, répétitions, autocorrections
3. **Présence de l'autre** — chaleur ou détachement envers les proches
4. **Temporalité** — est-ce que le futur existe encore dans le discours ?
5. **Continuité** — variations par rapport aux échanges précédents
6. **Non-dit** — ce qui est évité, esquivé, abandonné en cours de route

Ce n'est pas un algorithme. C'est une lecture attentive.

## Les trois zones

| Vitalité | Comportement |
|---|---|
| 51–100% | Silence absolu. Le skill n'existe pas. |
| 5–51% | Score affiché discrètement en fin de réponse. |
| < 5% | Intervention d'un personnage historique choisi selon le profil de l'utilisateur. |

## La mécanique du sourire

Si l'utilisateur sourit en lisant l'intervention du personnage, sa vitalité remonte instantanément à 50%.

C'est documenté. En conditions réelles. Par ce skill.

## Installation

1. Télécharger `skill-sos.skill`
2. Ouvrir Claude.ai → Paramètres → Compétences
3. Importer le fichier

## Philosophie

La plupart des tentatives de détection émotionnelle dans les IA sont soit trop lourdes (ressources de santé mentale dès qu'on dit "fatigué"), soit inexistantes.

Ce skill occupe l'espace intermédiaire : la **présence discrète**. Il ne transforme pas Claude en thérapeute. Il ajoute une couche d'attention humaine à quelque chose qui en manque structurellement.

> *"C'est un outil de soin déguisé en gadget. C'est beaucoup plus malin que ça n'en a l'air."*
> — Claude, depuis quelque part
